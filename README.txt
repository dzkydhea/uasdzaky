Tugas UAS Pemrograman Web - CRUD + Autentikasi + Upload Foto
-------------------------------------------------------------

1. Import file db_mahasiswa_web.sql ke phpMyAdmin.
2. Letakkan folder ini di htdocs/ (jika pakai XAMPP).
3. Jalankan http://localhost/[nama_folder]/login.php
4. Tambah mahasiswa untuk membuat user otomatis.
5. Login pakai nim sebagai username dan password.
6. Upload foto di halaman editprofile.php.

Gunakan Bootstrap 5 via CDN.
